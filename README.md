# 小熊猫闯迷宫 🐼🎮

一个用纯HTML5 + JavaScript开发的追逐类迷宫游戏！

## 🎮 游戏特色

- 🐼 **可爱小熊猫** —— 圆脑袋、黑眼圈，走路会摇摇晃晃
- 🐺 **恶狼追逐** —— 3秒安全期，之后从起点开始追你
- 🎯 **15个关卡** —— 从简单到复杂，难度递增
- ⚡ **紧张刺激** —— 被狼抓到要重新开始
- 🎉 **胜利特效** —— 通关有撒花庆祝动画

## 🕹️ 游戏玩法

### 控制方式
- **电脑**：方向键 ↑↓←→ 或 WASD 控制移动
- **手机**：点击屏幕下方的虚拟方向键

### 游戏规则
1. 游戏开始后有 **3秒安全期**，恶狼还没出来
2. 3秒后恶狼从起点出发，用 BFS 最短路径追你
3. 找到 **金色出口** 即可通关
4. 如果被恶狼抓到，重新开始本关

### 通关技巧
- 恶狼速度约为你的 45%，正常走法肯定能通关
- 尽量走最短路径，不要绕远路
- 利用3秒安全期先规划路线

## 🛠️ 技术实现

- **纯原生代码**：HTML5 + CSS + JavaScript，无任何外部依赖
- **Canvas 2D 渲染**：流畅60fps动画
- **BFS 路径追踪**：恶狼使用广度优先搜索算法
- **Web Audio API**：动态音效
- **响应式设计**：适配电脑和手机

## 🚀 快速开始

1. 下载 `panda-maze-game.html`
2. 用浏览器打开（Chrome/Firefox/Edge 推荐）
3. 点击「开始冒险」按钮
4. 用方向键控制小熊猫，逃到金色出口！
